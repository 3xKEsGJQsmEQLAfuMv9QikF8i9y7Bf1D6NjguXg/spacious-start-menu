# アンインストール

`Spacious Start Menu` を起動していたら終了しておきます。

## 💠手動インストールの場合

展開フォルダごと削除します。

## 💠Scoopを使用したインストールの場合

### アプリのアンインストール

以下のコマンドを実行します。

```
scoop uninstall spacious-start-menu
```

### bucketの削除

以下のコマンドを実行します。

```
scoop bucket rm 3xkesg
```
