# お勧め設定

タスクバーアイコンを左端の位置に移動します。

![Move icon](img/rcmd01.png)

タスクバーにピン留めします。

![Pin to taskbar](img/rcmd02-ja.png)

これで`Win` + `1`のショートカットキーでランチャーを起動またはアクティブにできるため、スタートメニューに近い呼び出し方ができます。
